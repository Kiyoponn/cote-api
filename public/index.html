Welcome to the Classroom of the Elite - API (v1)

Use these routes to request data on characters:

  • /characters - view all the characters
  • /character/id - view a character with the id, e.g., /charcters/3
  • /characters/students - view all the students
  • /characters/teachers - view all the teachers
